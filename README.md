## ü§î How To Run
1. Clone the project in local directory
2. Open the project in your preferred IDE, say VS Code
3. For VS Code: install the extension "Live Server"
4. Open the index.html file and run the live server by right clicking and selecting the option "Open with Live Server"
5. The project will run locally on your default browser at port 5500
